# ⏱️ Stopwatch Web App

A simple and responsive **Stopwatch Web Application** built using **HTML, CSS, and JavaScript**.  
This project allows users to start, pause, reset the timer, and record lap times.

## 🚀 Features

- Start and pause the stopwatch
- Reset the stopwatch
- Record lap times
- Accurate time display (Hours : Minutes : Seconds : Milliseconds)
- Responsive and clean user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
